Здравствуй, пользователь!


Для доступа к пользователям перейди на http://127.0.0.1:5000/users

Для доступа к конкретному пользователю перейди на http://127.0.0.1:5000/users/id пользователя

Для доступа к заказам перейди на http://127.0.0.1:5000/orders

Для доступа к конкретному заказу перейди на http://127.0.0.1:5000/orders/id заказа

Для доступа к предложениям перейди на http://127.0.0.1:5000/offers

Для доступа к конкретному предложению перейди на http://127.0.0.1:5000/offers/id предложения


Для осуществления POST/PUT/DELETE запросов я сохранил подготовленные мной запросы для postman. Они лежат в корневом руте приложения, просто добавь их в postman :)
